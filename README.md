4. TypeScript  
5. Docker  

## Run Instructions
1. **Setup Database and Mock Data**:  
   ```bash
   docker compose up -d
   ```

2. **Run Backend**:  
   ```bash
   air
   ```

3. **Run Frontend**:  
   ```bash
   yarn dev
   ```
