## Prerequisites
1. Golang  
2. Air CLI (for running Golang projects)  
3. TypeScript  
4. Docker  

## Run Instructions
1. **Setup Database and Mock Data**:  
   ```bash
   docker compose up -d
   ```

2. **Run Backend**:  
   ```bash
   air
   ```

3. **Run Frontend**:  
   ```bash
   yarn dev
   ```
